Reto #5

Para agregar un título a un documento HTML, se utiliza la etiqueta <title> dentro de la etiqueta <head>. El título es el texto que se muestra en la pestaña del navegador.

La descripción del documento se agrega con la etiqueta <meta> y el atributo name="description". Esta descripción es ideal para el SEO y se muestra en los resultados de búsqueda de Google.
