Reto #2

Para crear párrafos en HTML, se utiliza la etiqueta <p>. Esta etiqueta es un elemento de bloque, lo que significa que ocupa todo el ancho disponible.

Las imágenes, por otro lado, son elementos de línea. Esto significa que se muestran en línea con el contenido que los rodea.

Para insertar una imagen en una página, se utiliza la etiqueta <img>. Esta etiqueta no tiene una etiqueta de cierre porque no tiene contenido.
